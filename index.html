Abstract

MetaBatch is a database metadata analysis tool and Power BI dashboard for a Financial Institution. It automates the analysis of the database Metadata using Teradata SQL, for the Enterprise Data Warehouse team. It gathers information recursively about all dependencies of each view and table, and displays the findings on an interactive dashboard. The dashboard also shows summary timing information about the overnight batches, as well as the interdependencies between batches.
